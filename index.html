Points to Discuss

Open Issues & Decisions Needed
Architecture & Flow Issues
1 Application Mode Selection Conflict
Problem: We don't have retailers on the D2C panel, making "Application Mode" selection confusing.
Suggested Solution:
Use separate modules: "B2B Promo Management" and "D2C Promo Management" for promo creation instead of Application Mode dropdown.
2 Cashback Implementation Strategy
Scenario: For a ₹100 recharge with 10% cashback - first deduct ₹100, then credit ₹10 after successful transaction.
Current Status & Questions:
• B2B: Has wallet system - cashback works
• D2C: No wallet system (customers use Payment Gateway)
• Question: Implement wallet for D2C or use loyalty points?
User Targeting & Eligibility
3 User Eligibility Criteria
Requirement: Promo applicable to new users (first time use) or users who return after specific days.
Required Fields & Logic:
• User Type: New User / Existing User / All Users
• First Time Use: Boolean flag for new users only
• Days Since Last Transaction: Number field for returning users
• User Segment: Premium/Regular/All segments
4 Regional Promotions (Geo-specific Offers)
Example: Vaisakhi festival celebrated mainly in Punjab and Haryana.
Implementation Requirements:
• Add region/state selection in promo creation
• Integrate with user location detection
• Support inclusion/exclusion of regions
Technical Implementation
5 Promo Code Conflict Resolution
Issue: How to handle multiple applicable promo codes for a single transaction?
Proposed Solution:
• Implement priority levels for promo codes
• Allow only one promo per transaction
• Auto-select best offer for customer
6 Real-time Usage Tracking & Limits
Challenge: Ensuring accurate real-time tracking of usage limits.
Technical Considerations:
• Redis-based counters for real-time tracking
• Database fallback for consistency
• Rate limiting to prevent abuse
7 Promo Code Analytics & Reporting
Need comprehensive analytics to measure promo code effectiveness.
Required Metrics:
• Redemption rates by channel
• Customer acquisition cost
• Revenue impact analysis
• Regional performance breakdown
